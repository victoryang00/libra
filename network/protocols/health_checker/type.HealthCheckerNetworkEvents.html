type HealthCheckerNetworkEvents = NetworkEvents<HealthCheckerMsg>;

The interface from Network to HealthChecker layer.

HealthCheckerNetworkEvents is a Stream of PeerManagerNotification where the raw Bytes rpc messages are deserialized into HealthCheckerMsg types. HealthCheckerNetworkEvents is a thin wrapper around an channel::Receiver<PeerManagerNotification>.
