A map from byte code attribute to source code location.

annotations: Annotations

Annotations associated with this function.

Map of spec block ids as given by the source, to the code offset in the original bytecode. Those spec block's content is found at func_env.get_specification_on_impl(code_offset).

name_to_index: BTreeMap<Symbol, usize>

A map from local names to temp indices in code.

A cache of targets modified by this function.
